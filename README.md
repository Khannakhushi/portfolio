# Portfolio Website

A modern portfolio website built with Next.js, React, Tailwind CSS, and Framer Motion. This website showcases my professional experience, projects, leadership roles, and technical skills in an interactive and visually appealing format.

Visit the live site: [khyaatikhanna.com](https://khyaatikhanna.com)

## Features

- Responsive design
- Dark/light mode toggle
- Smooth animations with Framer Motion
- Section navigation
- Mobile-friendly interface

## Tech Stack

- **Framework**: Next.js 14
- **UI**: React with functional components
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Analytics**: Vercel Analytics
- **UI Components**: Radix UI primitives
